# Calculate R2 value of the scale-free model fit for the provided connectivities

Calculate R2 value of the scale-free model fit for the provided
connectivities

## Usage

``` r
scaleFreeFitRSquared(k)
```

## Arguments

- k:

  Numeric vector, the connectivities of a network.

## Value

Numeric, the \\R^2\\ value of the scale-free model fit.
